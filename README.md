# 🧬 Protein Generation Pipeline: KLIFS Dataset + Flow Matching

**Generación condicionada de estructuras proteicas de kinasas relacionadas con cáncer usando flow matching.**

---

## ⚡ Quick Start (2 min)

```bash
# 1. Setup
pip install -r requirements.txt

# 2. Download & process everything
make pipeline

# 3. (Optional) Explore data
jupyter notebook notebooks/dataset_visualization.ipynb

# 4. Train/Fine-tune
python training/finetune.py --config configs/default.yaml
```

**That's it.** You now have 500+ kinase structures in train/val/test splits, preprocessed and ready for fine-tuning.

---

## 📊 What This Pipeline Does

```
KLIFS API → Download 500+ kinase structures
           ↓
           Extract coordinates (Cα atoms)
           ↓
           Compute distance matrices
           ↓
           Group-stratified split (no leakage)
           ↓
           PyTorch tensors (ready for training)
           ↓
           FoldFlow conditioning (active/inactive state)
```

**Input**: KLIFS API (automatic)  
**Output**: Preprocessed dataset in `/data/processed/` + Train/Val/Test splits

## 🚀 Step-by-Step Execution

### Step 1: Setup (5 min)

```bash
# Create virtual environment (optional but recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Validate setup
python scripts/validate_setup.py
```

### Step 2: Download KLIFS Dataset (30-60 min)

```bash
# Automatically:
# - Downloads kinase structures from KLIFS API
# - Classifies as ACTIVE (DFG-in) or INACTIVE (DFG-out)
# - Creates grouped splits (no kinase overlap between train/test)
# - Generates metadata CSV

python scripts/download_klifs_dataset.py
# Or: make download-dataset
```

**Outputs:**
- `data/metadata/kinase_labels.csv` → All structures + labels + splits
- `data/raw/pdbs/*.pdb` → 500+ protein structures
- `data/splits/{train,val,test}.csv` → Split definitions (grouped by kinase)

### Step 3: Explore Dataset (2 min)

```bash
# See statistics
python scripts/explore_dataset.py
# Or: make explore-dataset

# Interactive visualization (in browser)
jupyter notebook notebooks/dataset_visualization.ipynb
```

**Output:**
```
📊 DATASET STATISTICS
- Total structures: 523
- Active: 343 (65.6%)
- Inactive: 180 (34.4%)
- Unique kinases: 13 (EGFR, BRAF, ABL1, ...)
- Avg resolution: 2.15 Å
```

### Step 4: Preprocess to PyTorch Tensors (15-30 min)

```bash
# Converts PDB files → PyTorch tensors
# Extracts Cα coordinates + distance matrices
# Saves for training

python scripts/preprocess_dataset.py
# Or: make preprocess
```

**Outputs:**
```
data/processed/[PDB_ID]/
├── ca_coords.pt           ← (N, 3) coordinates
├── distance_matrix.pt     ← (N, N) pairwise distances
└── metadata.pt            ← labels + metadata
```

### Step 5: Train/Fine-tune (N hours, depending on config)

```bash
# Fine-tune FoldFlow model with your data
python training/finetune.py --config configs/default.yaml

# Or customize:
python training/finetune.py \
  --config configs/default.yaml \
  --epochs 100 \
  --batch_size 32 \
  --learning_rate 1e-4
```

### Step 6: Evaluate

```bash
# Compute metrics (RMSD, diversity, etc.)
python evaluation/evaluate.py --checkpoint outputs/checkpoints/best.pt
```

## ⚙️ Configuration

Edit `config.yaml` to customize:

```yaml
# Kinases to download
kinases:
  - name: "EGFR"
  - name: "BRAF"
  - name: "ABL1"
  # ... 10 more cancer-related kinases

# Split strategy
splits:
  strategy: "grouped"        # "grouped" (recommended) or "stratified"
  group_by: "kinase_name"    # Group by kinase to prevent leakage
  train: 0.70
  validation: 0.15
  test: 0.15
  random_state: 42

# Preprocessing
preprocessing:
  extract_ca_only: true      # Use only alpha-carbon atoms
  compute_distance_matrix: true
  batch_size: 32

# Validation
validation:
  min_resolution: 0.5        # Angstroms
  max_resolution: 4.0
```

## 📦 Key Dataset Properties

| Property | Value | Note |
|----------|-------|------|
| **Source** | KLIFS API | Kinase structures only |
| **Total structures** | ~500 | Cancer-related human kinases |
| **Active/Inactive** | 65%/35% | Classified by DFG + alphaC states |
| **Split strategy** | Grouped | All EGFR in train OR val OR test (no overlap) |
| **Avg resolution** | 2.15 Å | Crystal structure quality |
| **Conformations** | DFG-in/out, alphaC-in/out | Reflects activation mechanism |

### Important: Data Leakage Prevention ⚠️

**This pipeline uses GROUP SPLITS (not random splits):**
- ✅ Train sees kinases: EGFR, BRAF, ABL1, AKT1, ...
- ✅ Val sees kinases: CDK4, KIT, MET, ...
- ✅ Test sees kinases: PDGFRA, FGFR1, ALK, ... (completely new)

**Why?** In production, you'll encounter kinases you never trained on. This setup measures real generalization, not memorization.

👉 **Learn more:** See `docs/LEAKAGE_AND_SPLITS.md` for technical explanation

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| `ModuleNotFoundError: requests` | `pip install -r requirements.txt` |
| `KLIFS API not responding` | Check internet. KLIFS might be down. Try later. |
| `Out of memory during preprocessing` | Reduce `batch_size` in `config.yaml` |
| `Test accuracy is much lower than validation` | Expected with grouped splits (honest measurement) |

👉 **More issues?** See `docs/TROUBLESHOOTING.md`
